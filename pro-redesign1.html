Redesign China Mobile app
This is a personal design practice. I tried redesigning China Mobile app by doing user research, then redesigned the interface according to the research and ux design principles.
User Research
China Mobile app is tool for China Mobile users to manage their phone package. Before redesigning, I interviewed 7 China Mobile users, and found that most of them use the app for almost the same things: check data usage, check account balance, check the bill, top up, etc. Their usage frequency of the app is as follows:
According to the research, 3 highest frequent usage of the app is to check data usage, check account balance and top up.

Home Page Redesign
Since "top up" is already on home page, but "data usage" and "account balance" are not shown directly on home page, I decided to add these information. Along with "data usage" and "account balance", I added "bill this month", assuming it's as important as the former 2. Service Handling Page Redesign
In this page, the original design is like the first screenshot in the following image, with lots of options in one page, resulting in small clickable area for every link.

In my design, the 2nd and 3rd interface, I tried to enlarge the clickable area, leaving only the most important links in this page. Version 1(the image in the middle) obviouly leave out too many options, may lead to decline in CTR. So I had version 2 (the 3rd image), which strikes a balance between conciseness and business goal.
Top-up Page Redesign
To top up, the original design takes 2 steps to complete, with lots of distracting compontents on every page. I combined the 2 steps into one, and enlarged the components that need to draw users' attention while reducing the size of others.
Campaign Page Motion Design
In the original design, there is a static header on the top, taking almost 1/6 of the screen. But the header is not as important as the content. So I decided to save screen space by motion design. As is shown in the following gif, the header expands when users pull down the page, and collapsed into only text when users begin to scroll up, thus saving screen space when users are browsing campaigns while showing icons on the header to guide them to go to other tab when they're trying to find more by pulling page down.
