Customize advertisement
This is a design exercise for Dianping’s ads platform where merchants can customize their ads display. The higher they set the ads fee, the higher probability the ads will show. This was designed in Chinese, and I translated it into English.
Background
1. Merchants’ advertisment fee is charged by the number of clicks. Merchants can set their own fee per click. The higher the fee per click, the higher the probability the advertisement will show.
2. Merchants have many different advertisement choices on Dianping app, such as “guess you like”, “search result list”, “other users also like”, etc. Merchants can set different fees for different advertisement display.
3. Merchants will choose a proper time period to show the advertisement according to their opening hours.
Demand Analysis
1. Merchants want the best ROI of their ads. They’d like to know how much ads exposure they will make by setting certain ads fee. They'll need a reference to set the fee.
2. Some merchants have only one store, some have dozens even hundreds of stores in a chain. We need to enable multi-selection of stores.
3. Some merchants’ stores open for a consecutive period in a day, some open for several separated periods in a day. They would need multi-selection of time periods.
Flow
Main user flow is like the image on the right.
When creating new ads, users need to set the stores, display area and display time.
Only when all the 3 elements are set can users submit.
Interaction Design
1. Ads creating and stores selecting
① Enable multi-selection of stores.
② Give users a reference to what prices other merchants pay for this type of ads, so they can make a better decision.
2. Ads display area seletion
① Give users a price reference before they fill in the ads fee field.
② Tell users how large the percentage of merchants they beat by setting this price. They can revise the fee accordingly.
3. Time period selection
① Enable multi-selection of time periods.
② Save the customized time periods for next time, so that users can select next time simply by clicking.
4. Submitting
① Use toast when submitted successfully.
② Users can edit or delete this ads after submitting.
