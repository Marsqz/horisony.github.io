Site Merchandiser Dashboard
Site Merchandiser Dashboard (SMD) is a self-service dashboard that helps eBay site merchandisers monitor eBay web pages in a more personalized way. By creating personalized page groups, site merchandisers can monitor both the overall performance of whole page group and detail performance of every single page.
