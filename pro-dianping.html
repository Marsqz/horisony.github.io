Coupon setting
Coupon setting is a function in Meituan-Dianping's marketing platform where shop owners can create, edit and delete coupons for their own shops.
Problem Analysis
1. In our previous coupon setting page, users can create only one type of coupon for one campagin. This doesn't meet the real requirement of shop owners.

2. Coupons can NOT be in common use in all shops in a chain. That doesn't meet the requirement of chain stores.

Below is the screenshot of our previous coupon setting page. I will translate them into English in my design below.
Design Goal
1. Enable users to create multiple coupons for one campagin.
2. Enable common use of coupon in chain stores.
Coupon Editing Design
1. Add "+ add coupons" button below coupon editing section so that users can add more coupons after creating one.
2. Add "Coupon Usage" single selection in basic settings, and choose "common use in all shops" by default. This will better suit chain stores' needs.
3. Add save and cancel buttons for every section so that users can save one section at one time.
4. Redesign the selection of campaign shops.
Coupon Information Display
If submitted successfully, show the information of the campaigns and coupons.
