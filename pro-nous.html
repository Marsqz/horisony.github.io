Nous Mobile App
Nous Mobile App is the mobile app version of Nous website which monitors the traffic data of eBay site. Mobile app enables users to check the traffic data of eBay site at any time and any place.
Web version of Nous
The web version of Nous was designed by another designer. I designed the mobile app version of Nous a few weeks later after the web version was released, according to users's feedback that they want to check the data on their mobile phone.
